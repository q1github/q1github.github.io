日语动词分类与变形规则

日语动词根据其活用（变形）规则主要分为三类：一类动词（五段动词）、二类动词（一段动词）、三类动词（カ変・サ変动词）。

一、动词分类识别方法

类别 别称 识别特征 常见词尾 示例
一类动词 五段动词 1. 不以「る」结尾
2. 以「る」结尾时，る前的假名在 あ段、う段、お段		 う、つ、る、ぬ、ぶ、む、く、ぐ、す 書く、立つ、帰る、死ぬ、遊ぶ、読む、話す
二类动词 一段动词 以「る」结尾，且る前的假名在 い段 或 え段 いる、える 見る、食べる、起きる、寝る
三类动词 变格动词 极其不规则 する、来る する、勉強する、来る（くる）

注意区分一类动词中的「る结尾」动词：一类动词词尾る前的假名在 あ段、う段、お段（如 帰る「かえる」、走る「はしる」、知る「しる」）；二类动词词尾る前的假名在 い段 或 え段（如 見る「みる」、食べる「たべる」）。

二、动词基本变形规则表（常用形）

变形类型 一类动词 (五段) 示例：書く（かく） 二类动词 (一段) 示例：食べる（たべる） 三类动词 (カ変) 来る（くる） 三类动词 (サ変) する / 勉強する
辞書形 書く 食べる 来る（くる） する / 勉強する
ない形 書かない 食べない 来ない（こない） しない / 勉強しない
ます形 書きます 食べます 来ます（きます） します / 勉強します
て形 書いて ① 食べて 来て（きて） して / 勉強して
た形 書いた ① 食べた 来た（きた） した / 勉強した
ば形 書けば 食べれば 来れば（くれば） すれば / 勉強すれば
意向形 書こう 食べよう 来よう（こよう） しよう / 勉強しよう
命令形 書け 食べろ / 食べよ ② 来い（こい） しろ / せよ / 勉強しろ
可能形 書ける 食べられる 来られる（こられる） できる / 勉強できる
受身形 書かれる 食べられる 来られる（こられる） される / 勉強される
使役形 書かせる 食べさせる 来させる（こさせる） させる / 勉強させる

变形规则说明：

  1. **一类动词て形/た形音便规则 (重要！)**：
    • 词尾 う、つ、る → って / った (買う→買って/買った、立つ→立って/立った、帰る→帰って/帰った)
    • 词尾 ぬ、ぶ、む → んで / んだ (死ぬ→死んで/死んだ、遊ぶ→遊んで/遊んだ、読む→読んで/読んだ)
    • 词尾 く → いて / いた (書く→書いて/書いた) 例外：行く（いく）→ 行って / 行った
    • 词尾 ぐ → いで / いだ (泳ぐ→泳いで/泳いだ)
    • 词尾 す → して / した (話す→話して/話した)
  2. 二类动词命令形：口语常用「〜ろ」 (食べろ)，书面语常用「〜よ」 (食べよ)。
  3. 三类动词サ変的可能形：直接用「〜できる」(勉強できる)，不是「〜するできる」。

三、各类动词详细说明与用例

1. 一类动词 (五段动词)

  • 特征：词尾在あいうえお五段上变化。
  • 辞書形：話す、読む、行く、泳ぐ、待つ

    例：本を読む。(读书。)

  • ない形：話さない、読まない、行かない、泳がない、待たない

    例：彼はあまり話さない。(他不怎么说话。)

  • て形：話して、読んで、行って、泳いで、待って (注意音便规则！)

    例：図書館へ行って、本を読んで、帰る。(去图书馆看书然后回家。)

  • 可能形：話せる、読める、行ける、泳げる、待てる (词尾う段→え段 + る)

    例：日本語が話せる。(会说日语。)

2. 二类动词 (一段动词)

  • 特征：变形规则简单一致，去掉词尾「る」后加各种活用后缀。
  • 辞書形：見る、起きる、教える、寝る

    例：毎朝6時に起きる。(每天早上6点起床。)

  • ない形：見ない、起きない、教えない、寝ない (去る + ない)

    例：テレビを見ない。(不看电视。)

  • ます形：見ます、起きます、教えます、寝ます (去る + ます)

    例：先生が英語を教えます。(老师教英语。)

  • 可能形：見られる、起きられる、教えられる、寝られる (去る + られる)

    例：この窓から富士山が見られる。(从这个窗户能看到富士山。)

3. 三类动词 (カ変・サ変)

  • カ変：来る（くる）
    • 辞書形：来る

      例：明日、友達が来る。(明天朋友来。)

    • て形：来て（きて）

      例：早く来てください。(请快点来。)

    • ない形：来ない（こない）

      例：彼は今日は来ない。(他今天不来。)

  • サ変：する / 〇〇する
    • 辞書形：する、勉強する、旅行する

      例：毎日勉強する。(每天学习。)

    • ない形：しない、勉強しない、旅行しない

      例：週末は仕事をしない。(周末不工作。)

    • 可能形：できる、勉強できる、旅行できる (特殊！)

      例：一人で旅行できる。(能一个人旅行。)

四、总结对比与学习要点

特性 一类动词 (五段) 二类动词 (一段) 三类动词 (カ変・サ変)
规则性 较复杂，需记音便 (尤其て/た形) 最规则，活用一致 最不规则，需单独记忆
数量 最多 较多 只有「来る」和「する」及复合词
核心变化 词尾在あいうえお五段变化 去「る」加活用后缀 カ変：く→こ→き / サ変：す→し→せ
学习重点 掌握て形/た形音便规则、可能形变化 熟练去掉「る」加后缀 背诵「来る」「する」所有变形

学习建议：

  1. 先掌握二类动词：规则最简单，建立信心。
  2. 重点攻克一类动词て形/た形：音便规则是难点也是重点，多练习常用动词（行く、買う、話す、飲む、読む、待つ等）。
  3. 死记硬背三类动词：把「来る」和「する」的所有变形当作固定搭配背诵。复合サ変动词（〜する）按「する」的规则变形。
  4. 多听多读多用：在实际语境中感受和运用动词变形。
